HTML and CSS Documentation
Introduction

HTML and CSS are known as "frontend" languages.

This means that you can directly see everything, and although you can't see the code itself, these two languages power the look and feel of a webpage.

What You Should Know

HTML (short for "HyperText Markup Language") is the most basic building block of the web, since it defines the meaning and structure of web content.

CSS (short for "Cascading Style Sheets") describes the web page's appearance, which makes the web pages look good.

What is HTML?

HTML can be seen as the "skeleton" of the web page, and this gives the web page some structure.

The use of tags can be used to define the structure of the page, and without tags, the website wouldn't be able to function properly.
